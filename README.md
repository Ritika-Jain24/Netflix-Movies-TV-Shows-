# 🎬 Netflix Movies & TV Shows Analysis

An end-to-end Data Analytics project that analyzes the Netflix Movies & TV Shows dataset using **Python**, **MySQL**, and **Power BI**. This project demonstrates the complete analytics workflow, from data cleaning and preprocessing to SQL analysis and interactive dashboard creation.

---

## 📌 Project Overview

The objective of this project is to explore the Netflix content library and extract meaningful insights regarding:

- Movies vs TV Shows distribution
- Content ratings
- Most active countries
- Top directors
- Popular genres
- Release trends
- Average movie duration
- Monthly content additions

## 📈 Key Insights

- 🎥 Movies dominate the Netflix catalog.
- 🌍 United States has the highest number of titles.
- ⏱️ Average movie duration is approximately **106 minutes**.
- 📅 December records the highest number of content additions.
- 🎭 Drama is one of the most popular genres.
- ⭐ TV-MA is the most common content rating.
